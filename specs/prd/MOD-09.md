# MOD-09 — Radicación de producto final
> Parte del PRD. Ver índice en `specs/PRD.md`.

**Descripción:** Entrega formal del producto final del trabajo de grado.
**Actores:** Estudiante.

| ID | Requerimiento | Actor |
|---|---|---|
| RF-09-01 | El sistema permite radicar el producto final solo cuando el trabajo está en estado **En desarrollo** y hay una ventana de fechas activa para radicación de producto final (o ventana extemporánea habilitada) | Estudiante |
| RF-09-02 | El sistema exige los siguientes adjuntos obligatorios para todas las modalidades: (a) producto final en la plantilla correspondiente a la modalidad, (b) carta de aval del director, (c) reporte de similitud | Estudiante |
| RF-09-03 | Para la modalidad **Innovación y Emprendimiento**, el sistema exige un cuarto adjunto obligatorio: certificación de inscripción del Plan de Negocio ante fondo de financiación reconocido o unidad de innovación USC | Estudiante |
| RF-09-04 | Si el trabajo está vinculado a una empresa u organización, el sistema permite adjuntar la carta de impacto emitida por la empresa (adjunto condicional, validado posteriormente por el Administrador) | Estudiante |
| RF-09-05 | El sistema no permite confirmar la radicación si algún adjunto obligatorio está faltante | Sistema |
| RF-09-06 | Al confirmar la radicación, el sistema cambia el estado a **Producto final entregado** y notifica al Administrador | Sistema |
