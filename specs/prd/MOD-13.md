# MOD-13 — Generación de acta
> Parte del PRD. Ver índice en `specs/PRD.md`.

**Descripción:** Cierre del proceso con la autorización de publicación y la emisión del acta oficial.
**Actores:** Administrador, Estudiante.

| ID | Requerimiento | Actor |
|---|---|---|
| RF-13-01 | Una vez el trabajo está en estado **Trabajo aprobado**, el sistema habilita al estudiante para diligenciar el formato de autorización de publicación para la biblioteca (indicando si autoriza o no) | Estudiante |
| RF-13-02 | El Administrador puede emitir el Acta de Sustentación / Aprobación de Trabajo de Grado una vez el estudiante ha diligenciado la autorización | Administrador |
| RF-13-03 | El acta queda registrada en el sistema asociada al trabajo de grado | Sistema |
| RF-13-04 | Al emitir el acta, el sistema cambia el estado a **Acta generada** | Sistema |
| RF-13-05 | El estudiante puede descargar el acta desde el sistema | Estudiante |
